## 配置key
- 切换到Claude,点击Add Provider
- ![img_2.png](img_2.png)
- 输入Provider Name(自定义)
- API Key为第一步创建apikey复制的令牌
- API Request URL:https://apisstore.com
  ![img_3.png](img_3.png)
- 填写完整后打开终端输入claude,即可对话
