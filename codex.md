# GPT-codex 安装教程
- 下载cc switch,下载链接:
- windows: [cc switch](https://github.com/Changochen123/cc-switch/releases)
- mac: [cc switch](https://github.com/Changochen123/cc-switch/releases)
## 安装codex
- npm i -g @openai/codex
## 配置key
- 点击Add Provider
- ![img.png](img.png)
- 输入Provider Name(自定义)
- API Key为第一步创建apikey复制的令牌
- API Request URL:https://apisstore.com/v1
- 选择模型为gpt-5.4
- 填写完整后打开终端输入codex,即可对话
